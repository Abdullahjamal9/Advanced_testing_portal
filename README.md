# 🧪 Advanced Testing Portal

An enterprise-level **Testing, Evaluation, and Certification Management System** designed to automate employee assessments, manage standards-based exams, handle practical evaluations, and generate dynamic certificates using templates.

---

## 🚀 Key Highlights

- 🧑‍💼 Dual-role system (Admin + Employee)
- 📝 Online testing with skip/review functionality
- 📊 Admin analytics dashboard with charts and stats
- 🧾 Automated certificate generation system
- 📚 Standards-based evaluation framework
- 🧪 Theory + Practical combined scoring system
- 🤖 Auto-grading system with negative marking (0.25)
- 📎 Attachment-based result verification
- ❓ Bulk question management system

---

## 📌 Overview

The Advanced Testing Portal is a full-stack web application developed to streamline and automate employee assessment workflows.

The system enables organizations to:
- Conduct online examinations
- Automatically evaluate results
- Manage standards and certifications
- Handle practical assessments
- Generate dynamic certificates
- Store answer sheets and supporting documents

---

## 👤 Employee Module

### Features
- Login using Employee ID
- Auto-fetch employee details
- Select standard/subject
- Start online test
- Skip and revisit questions
- Submit final answers
- Automatic result evaluation

---

## 🤖 Auto-Grading System

- Automatic answer evaluation
- Negative marking support (0.25 deduction per wrong answer)
- Dynamic passing percentage defined per standard
- Automatic pass/fail determination
- Result calculation without manual intervention

---

## 🛠️ Admin Module

### 📊 Dashboard
- Interactive analytics dashboard
- Statistics cards
- Charts and graphs
- Top 10 recent results table
- System-wide performance overview

### 📄 Results Management
- Filter by name, employee ID, status, standard, and date
- Download answer sheets
- Edit or delete results
- Manually add results
- Upload external result attachments

### 🧪 Practical Evaluation System
- Identify candidates eligible for practical tests
- Add practical examination results
- Combine theory + practical evaluations
- Determine final certification eligibility

### 📚 Standards Management
- Add/edit/delete standards
- Set passing percentage
- Assign certificate templates
- Certificate number generated from template filename

### 🎓 Certificate Generation System
- New certification & recertification support
- Dynamic certificate generation
- Automatic certificate numbering
- Previous certification tracking

### ❓ Question Bank Management
- Bulk upload/download/delete
- Add single questions
- Edit individual questions
- Advanced filtering support

### 👥 Employee Management
- Add/edit/delete employees
- Maintain employee database

### 📎 Attachment System
- Upload supporting documents
- Download attachments later for audit/reference

---

## ⚙️ System Workflow

```text
Employee Login
      ↓
Select Standard
      ↓
Start Online Test
      ↓
Answer Submission
      ↓
Auto Grading Engine
      ↓
Result Evaluation
      ↓
Practical Evaluation (if required)
      ↓
Certificate Generation
```
